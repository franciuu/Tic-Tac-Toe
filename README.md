# Tic-Tac-Toe Game

<p>This project represents a Tic-Tac-Toe game implementation, developed using HTML, CSS, and JavaScript. The project was created as part of the <strong>100 Days Of Code - 2024 Web Development Bootcamp</strong> and aims to practice essential web development concepts such as interactivity and DOM manipulation.</p>

## Features
<ul>
  <li>Responsive design: The game is suitable for different screen sizes, both on desktop and mobile devices.</li>
  <li>Two-player mode: Allows alternation between two players using the symbols "X" and "O".</li>
  <li>Winner detection: The game logic automatically identifies the winner or declares a tie, depending on the situation.</li>
  <li>Reset functionality: Allows resetting the game to start a new round.</li>
</ul>
